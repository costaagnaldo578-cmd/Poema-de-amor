# Poema-de-amor
No silêncio que antecede o toque, teu nome é a primeira palavra. Não a que sai da boca, mas a que habita o osso, a que pulsa na curva do pescoço como um segredo que o corpo inteiro guarda.  Teu riso é um fio de água onde eu bebo a minha própria sede. Teus olhos, dois pequenos incêndios que queimam sem consumir, e eu, bobo.
